# 将棋プレミアム (Shogi Premium)

ブラウザで快適に遊べる、美しくレスポンシブな和モダン将棋ゲームです。
パソコンとスマートフォンの両方に最適化された操作性とデザインを備えています。

## 🌟 特徴
- **オンライン対戦 (VS PLAYER)**: Firebase Realtime Database を使用し、ルームID（6桁）を入力するだけで離れた相手とリアルタイムに対戦できます。
- **コンピュータ対戦 (VS COM)**: Web Worker を使用した非同期将棋思考エンジンを搭載。ブラウザの描画をブロックせず、スムーズに動作します（難易度3段階）。
- **プレミアムな和モダンデザイン**: 漆塗り風の盤面と、ネオン調に光る駒を組み合わせたサイバー和風デザイン。
- **サウンド演出**: Web Audio API によるリアルな「パチッ」という駒の打音。
- **完全レスポンシブ**: パソコンではドラッグ＆ドロップ、スマホではタップによる直感的な操作が可能です。
- **反則・ルール判定**: 二歩、行き所のない駒、打ち歩詰め、王手放置など、将棋の反則ルールを完全にカバー。

## 🚀 はじめ方
1. `index.html` をブラウザで直接開くか、Webサーバーでホスティングします。
2. オンライン対戦を使用する場合は、[FIREBASE_SETUP.md](FIREBASE_SETUP.md) の手順に従って Firebase Realtime Database を設定してください。

## 📁 ファイル構成
- `index.html`: ゲーム画面、コントロールパネル、モーダルなどのHTML
- `styles.css`: レスポンシブレイアウト、ネオン和風テーマのCSS
- `app.js`: 将棋の駒の動き、反則判定、UI・音響・Firebase制御ロジック
- `ai-worker.js`: バックグラウンドで動作する Alpha-Beta 探索AI
- `FIREBASE_SETUP.md`: Firebaseの設定手順書

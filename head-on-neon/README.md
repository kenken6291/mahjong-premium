# HEAD-ON NEON

ブラウザで軽快に動作する、レトロな音付きの「ヘッドオン（Head-On）」風レーンチェンジ・アクションゲームです。
1970年代のクラシックなゲーム性をベースにしつつ、80s Synthwave風のネオンカラーとWeb Audio APIによるレトロな8bitサウンドエフェクトでリメイクしました。

## 特徴
- **レトロな8bitサウンド**: 外部音源ファイルを一切読み込まず、Web Audio APIだけでシンセサイズされた効果音（リアルタイムにピッチが変わるエンジン走行音、ピコピコとしたドット回収音、スライド音、重低音ノイズ爆発によるクラッシュ音、ステージクリアのファンファーレ）を再生します。
- **美しいネオンデザイン**: CSSとCanvasによる発光エフェクト（Glow効果）を駆使した、サイバーパンク/レトロフューチャーなグラフィック。
- **マルチデバイス対応**: PCのキーボード操作はもちろん、スマートフォンでの親指操作に最適化されたタッチパネルを搭載。
- **段階的な難易度**: ステージが進むにつれて敵車（赤いビークル）が増え、敵のAIがより積極的にプレイヤーを追いかけるようになります。

## プレイ方法

### PC操作
- **車線変更**: 矢印キー（`↑` `↓` `←` `→`）または `W` `A` `S` `D`
  - 現在走っている辺と、変更したい車線（内側/外側）に合わせて直感的に押します。
  - 例：上辺を左に走っている時：`↓`（または `S`）で内側へ、`↑`（または `W`）で外側へ車線変更。
- **加速 (アクセル)**: `スペースキー` または `左シフトキー`
- **音声オン/オフ**: 画面右下のスピーカーアイコンをクリック。

### スマートフォン操作
- **内側へ車線変更**: `IN` ボタンをタップ（または押しっぱなし）
- **外側へ車線変更**: `OUT` ボタンをタップ（または押しっぱなし）
- **加速**: `ACCEL` ボタンを押しっぱなし

## ローカルでの実行方法

このディレクトリ内の `index.html` をブラウザで直接開くか、ローカルサーバーを起動して実行してください。

例 (Python 3):
```bash
python -m http.server 8000
```
その後、ブラウザで `http://localhost:8000` にアクセスします。

## GitHubへの公開手順

このプロジェクトをGitHubリポジトリに公開し、GitHub Pagesを利用してWeb上にデプロイする手順は以下の通りです。

### 1. ローカルのGit初期化
本フォルダ内で以下のコマンドを実行します。
```bash
git init
git add .
git commit -m "Initial commit: Add HEAD-ON NEON game files"
```

### 2. GitHubにプッシュ
GitHubで新しい空のリポジトリ（例：`head-on-neon`）を作成し、表示されるコマンドを実行してプッシュします。
```bash
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/head-on-neon.git
git push -u origin main
```

### 3. GitHub Pages で公開する
1. GitHub上のリポジトリページを開きます。
2. **Settings**（設定）タブをクリックします。
3. 左メニューから **Pages** を選択します。
4. **Build and deployment** の Source にて `Deploy from a branch` を選択します。
5. Branch に `main`（または `/ (root)`）を選択し、**Save** をクリックします。
6. 数分待つと、`https://あなたのユーザー名.github.io/head-on-neon/` でゲームが一般公開されます。

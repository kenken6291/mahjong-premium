# Gunjin Shogi Premium (軍人将棋 プレミアム)

ブラウザで軽快に遊べる、通信対戦およびPC（AI）対戦機能を搭載したプレミアムな軍人将棋ゲームです。
洗練されたミリタリー、ネオン、クラシックの3つのデザインテーマと、Web Audio APIによる本格的な効果音で、臨場感あふれる対局を楽しめます。

---

## 🎮 ゲームの特徴

1. **非対称情報の戦闘システム**
   - 相手の駒の正体は伏せられたまま（裏向き）で進行します。
   - 駒が衝突した時、自動審判システムが勝敗を判定し、生き残った駒は裏向きのまま盤上に残ります。
   - 戦闘で撃破された駒は、双方の「駒台（墓地）」に表向きで公開され、お互いに過去の戦果を確認・推理できます。

2. **3つの対戦モード**
   - **COM対戦 (VS PC)**: 不完全情報に対応したAI思考エンジンと戦います。AIは対局履歴からプレイヤーの駒の正体を確率的に推測して行動します。
   - **通信対戦 (Online)**: Firebase Realtime Database を利用し、ルームIDを共有するだけで簡単に遠隔のプレイヤーと対戦できます。
   - **ローカル対戦 (VS Friend)**: 1台のPCやスマートフォンを交互に操作し、対面で対戦します（自分の操作時以外は、相手の操作が隠されるプライバシー設計）。

3. **自由度の高い配置フェーズ**
   - 対局開始前に、自分の駒18枚を自陣の3段（18マス）に自由に配置可能。
   - 軍旗を司令部に隠し、地雷で守るか、大将を最前線に置くかなど、戦略の幅は無限大。
   - ワンタップで配置できる「ランダム配置」や「配置クリア」もサポート。

4. **マルチデバイス対応**
   - パソコン、タブレット、スマートフォンのいずれの画面サイズにも最適化されたレスポンシブデザイン。
   - スマホではタップ操作、PCではドラッグ＆ドロップおよびタップ操作の双方で直感的に操作可能。

5. **3つのビジュアルテーマ**
   - **Military (ミリタリー)**: デジタル迷彩、スチールグレー、オリーブドラブを多用した重厚感ある軍事テイスト。
   - **Neon (ネオン)**: サイバーパンク調のダークな闇に鮮やかなネオンカラーが浮かび上がるフューチャリスティックなテイスト。
   - **Classic (クラシック)**: 伝統的な将棋盤と木製駒を模した、落ち着いた和風テイスト。

## 📐 ルール概要

### 盤面構成
- **サイズ**: 縦8マス × 横6マス。
- **キャンプ (安全地帯)**: 敵味方それぞれ4箇所（合計8箇所）。ここにある駒は攻撃されません。また、キャンプからは1マスしか移動できません。
- **司令部**: 最下段の左右2マス。一度入った駒は移動できなくなります。敵の司令部を占領することが勝利条件の1つです。
- **川と橋**: 中央の4段目と5段目の間にある川。通常の駒は「橋」（縦・斜めの一部ルート）を通ってのみ渡れます。※工兵と飛行機は川を無視して渡れます。

### 駒の戦闘力関係
- 基本は階級の強さ順： **大将 > 中将 > 少将 > 大佐 > 中佐 > 少佐 > 大尉 > 中尉 > 少尉** (数字が大きい方が勝ち、同階級は相打ち)。
- **スパイ**: 最強の「大将」を暗殺可能（大将に勝つ）。それ以外のすべての戦闘駒・地雷には負ける。
- **工兵**: 地雷を無傷で撤去可能（地雷に勝つ）。スパイと軍旗にも勝つが、それ以外のすべての駒に負ける。
- **飛行機**: 移動力に優れる特殊駒（駒を飛び越えて直線移動）。少尉以下、地雷、軍旗、スパイに勝つ。中尉以上には負ける。
- **突撃兵**: 歩兵クラスの駒。工兵、スパイ、軍旗に勝つ。少尉以上の軍人、飛行機、地雷には負ける。
- **地雷**: 移動不可。攻撃してきた駒（工兵以外）とすべて相打ち（両方消滅）になる。工兵に攻撃されると一方的に消滅する。
- **軍旗**: 移動不可。戦闘力はなく、攻撃されると即座に敗北。必ず司令部（左右どちらか）に配置する。

### 勝利条件
1. 敵の「軍旗」を占領する（敵の軍旗が配置された司令部に、自分の少尉以上の駒が突入し勝利する）。
2. 相手の動かせる駒（戦闘力のある駒）を全滅させる。

## 🛠️ セットアップと起動方法

1. リポジトリをクローンまたはダウンロードします。
2. オンライン通信対戦を利用する場合は、[FIREBASE_SETUP.md](file:///c:/Users/user/Antigravity/gunjin-shogi-premium/FIREBASE_SETUP.md) を参考にしてFirebaseの設定を `app.js` の先頭に貼り付けます。
3. ローカルサーバーを起動するか、`index.html` を直接ブラウザで開くだけでプレイが可能です（推奨：`live-server` などの静的ファイルサーバー）。

## 📂 技術構成
- **コア**: HTML5 / CSS3 (Vanilla CSS) / ES6+ JavaScript
- **通信同期**: Firebase Realtime Database SDK
- **AIエンジン**: Web Worker (マルチスレッドでの非同期探索)
- **音響効果**: Web Audio API (動的オシレータによる効果音生成、アセットロード不要)

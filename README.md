# Mahjong Premium (オンライン4人麻雀ゲーム)

高品質なビジュアルデザインと自動点数計算、COM（AI）自動補填、さらにリアルタイム通信対戦を搭載した、ブラウザで動作するプレミアムな4人麻雀ゲームです。

---

## 🌟 特徴
- **2つのプレイモード**:
  - **オンライン対戦 (4人)**: ルームIDを共有することで、遠隔地のプレイヤー同士で最大4人でリアルタイムに対戦できます。人数が不足している場合は、自動的にコンピュータ（COM）が空きスロットに入ります。
  - **練習戦 (VS コンピュータ)**: オフラインでCOM3人と手軽に対戦練習ができるモードです。
- **赤ドラルール対応**:
  - ルール設定により、赤ドラ（赤五萬、赤五筒、赤五索を各1枚、計3枚）の有無を指定できます。
- **自動点数計算 ＆ 自動役判定**:
  - アガリ時の面子分解（DFSによる自動判定）から、立直、断幺九、平和、ホンイツ、チンイツ、ドラ、赤ドラといった各種役を自動検知します。
  - 符計算（待ち方や面子の種類による加符）も自動で算出し、正しい点数授受が行われます。
- **プレミアムなUI/UXデザイン**:
  - **3D麻雀卓**: 深いエメラルドグリーンのラシャ（布）を表現した美しい3D調の対戦画面。
  - **立体麻雀牌**: ホバー時に浮かび上がる3D効果と美しい影を完備した麻雀牌。赤ドラは赤く輝く光沢エフェクト。
  - **カットイン演出**: リーチやアガリ（ツモ/ロン）発生時に、ネオンや筆文字を用いた迫力あるカットインが表示されます。
  - **臨場感のあるサウンド**: Web Audio APIを用いた打牌時の硬質な電子打音を動的に合成して再生（外部音声ファイルの読み込み不要）。
- **レスポンシブ対応**:
  - PCの大画面からスマートフォンの縦長・横長画面まで、比率を維持したまま自動で最適な大きさにスケールします。

## 🚀 起動方法

本アプリケーションは HTML, CSS, JavaScript のみで構成されているため、特別なサーバー側のインストールは不要です。

1. このフォルダ内の `index.html` をダブルクリックするか、ブラウザにドラッグ＆ドロップします。
2. もしくは、ローカルサーバー（VS CodeのLive Server、または Python の `python -m http.server 8000` など）を起動してアクセスします。

## 🛠️ GitHubへの登録とインターネット公開手順

このゲームをGitHubにアップロード（登録）し、さらに誰でも遊べるようにインターネット上に公開（GitHub Pages）する手順です。
※ ユーザー名は `kenken6291` を使用します。

### 1. GitHub上で新しいリポジトリを作成
1. [GitHub](https://github.com/) にログインします。
2. 画面右上の「**+**」アイコンをクリックし、「**New repository**」を選択します。
3. **Repository name** を `reversi-premium` (または新規の `mahjong-premium` など) にします。
4. 公開範囲を「**Public**（パブリック）」に設定します。（インターネット公開に必要です）
5. 「Initialize this repository with:」の項目はすべて**チェックを入れずに空欄のまま**にします。
6. 「**Create repository**」ボタンをクリックします。

### 2. リポジトリをGitHubに接続してプッシュする
GitHubリポジトリ作成後に表示されるコマンドを実行します。

```bash
# 1. ブランチ名を main に設定
git branch -M main

# 2. リモートリポジトリ（GitHub）の登録
git remote add origin https://github.com/kenken6291/reversi-premium.git

# 3. GitHubへソースコードをアップロード
git push -u origin main
```

### 3. インターネット上に無料公開する（GitHub Pages）
1. GitHubのリポジトリページにアクセスします。
2. 上部タブの「**Settings**（設定）」をクリックします。
3. 左メニューの「Code and automation」カテゴリにある「**Pages**」をクリックします。
4. 「Build and deployment」の中の **Source** を「**Deploy from a branch**」にします。
5. **Branch** で `main` を選択し、「**Save**」をクリックします。
6. 数分待つと、画面上部に公開されたURL（例: `https://kenken6291.github.io/reversi-premium/`）が表示され、友人と通信対戦ができるようになります！
